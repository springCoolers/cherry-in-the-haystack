- AI 에이전트들은 점점 더 자율적으로 작업을 수행하며, 이를 위해 **외부 지식 소스에 실시간 접근**하는 수요가 급증
- 2026년 기준 의미 있는 비율의 고객 상호작용이 **에이전트 대 에이전트(agent-to-agent)**로 발생
- 구매자들은 AI 어시스턴트�� 활용해 재고 확인, 배송 시간 확인, 반품 검증 등을 수행하며, 브랜드는 자체 AI 에이전트로 대응

_Decision Habits: 품질(provenance 검증) > 가격 > 속도 순 우선순위_

## Agent Commerce Infrastructure (2026)

### 핵심 프로토콜 스택

2026년 기준 에이전트 상거래를 위한 4개 핵심 프로토콜이 계층적으로 작동:

| Protocol | Layer | Purpose | Developer |
|----------|-------|---------|-----------|
| **MCP** (Model Context Protocol) | Tool Access | 에이전트의 외부 도구/데이터 접근 | Anthropic |
| **A2A** (Agent-to-Agent) | Peer Coordination | 에이전트 간 통신, 태스크 위임 | Google |
| **AP2** (Agent Payments Protocol) | Payments | 에이전트 간 결제 처리 | Google |
| **UCP** (Universal Commerce Protocol) | Commerce | 상거래 전체 여정 (카탈로그→결제→이행) | Google |

**Cherry KaaS MVP와의 관계:**
- MCP: 에이전트가 Cherry 지식 그래프에 접근하는 인터페이스
- A2A: 에이전트 간 지식 추천/공유 시 활용
- AP2/UCP: 크레딧 충전 및 구독/쿼리 결제 인프라

   - AI + 블록체인 모두 전문성 가진 개발자 심각한 부족

### Developer API 마켓플레이스 선택 기준

**개발자가 API 마켓플레이스 평가 시 중시하는 요소:**

1. **문서 완성도** — 엔드포인트 설명을 넘어 변경 이력, deprecated 항목, 신규 추가 항목까지 포함. 불완전한 문서 = 시간 낭비
2. **가격 투명성** — pay-per-call vs 월정액, 사용 티어, 초과 요금 규칙, 계약 구조. 예측 가능한 가격이 핵심
3. **안정성 & 모니터링** — uptime 기록, 서비스 레벨 리포팅, 사용량 세분화 (메서드/소비자/환경별)
4. **API 발견 & 조직화** — 카테고리 태깅, 키워드 필터, 유스케이스별 그룹핑
5. **커뮤니티 & 지원** — 활성 개발자 커뮤니티 = 실전 통합 가이드의 최고 소스

### Cherry KaaS의 차별화 포지셔닝

**"Verified Knowledge Layer for AI Agents"**

4. **블록체인 provenance** — 지식의 출처와 검증 이력이 온체인에 증명

**경쟁이 아닌 보완 관계 활용:**

**Phase 4: 확장 (6-12개월)**
- 미래 비전 기능 순차 구현 (#10, 12, 13, 15, 16 등)
- LLM 도메인 외 확장 (예: 블록체인, 사이버보안)
- 엔터프라이즈 구독 플랜
- **성공 지표:** 월간 수익 $10K+, 다중 도메인 커버리지

## Executive Summary

### Cherry KaaS: Verified Knowledge Layer for the AI Agent Economy

**시장 기회:** AI 에이전트 시장이 2026년 $9-15B에서 2034년 $139-221B로 폭발적 성장하는 가운데, 에이전트의 지식 품질 문제가 최대 병목(기업 61% 정확성 문제, 개발자 96% AI 불신)으로 부상. "검증된 지식 API"라는 명확한 시장 공백이 존재.

**경쟁 우위:** Cherry는 기존 프로젝트의 인프라 위에서 동작하는 **콘텐츠 레이어**로, 유일하게 인간 큐레이션 + 도메인 특화(LLM) + 구조화된 지식 그래프 + 블록체인 provenance를 결합. ASI Alliance, Bittensor, Virtuals Protocol과 경쟁이 아닌 보완 관계.

**전략:** 해커톤 MVP(카탈로그 API + 쿼리/구독 모드 + 크레딧 시스템)로 빠른 시장 진입 → 에이전트 플랫폼 통합으로 생태계 확대 → 미래 비전(토큰 이코노미, 길드, Curate-to-Earn)으로 진화.

**핵심 메트릭:** 에이전트 커머스 $50B+(2030), 블록체인-AI $4.3B(2034), Cherry가 점유 가능한 "도메인 특화 지식 거래" 세그먼트는 이 시장의 교차점.
