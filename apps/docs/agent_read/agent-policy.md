# Agent 작업 정책

## 1. 작업 로그 필수 작성

작업을 수행할 때마다 반드시 작업 로그를 작성한다.

- 로그 파일 위치: `apps/docs/agent_read/work-log-YYYY-MM-DD.md`
- 날짜별 하나의 파일. 같은 날 여러 작업이 있으면 같은 파일에 항목을 추가한다.

---

## 2. 본 프로젝트에서 직접 작업

워크트리(`\.claude\worktrees\`)에서 작업하지 않는다.  
모든 파일 작업은 본 프로젝트 경로(`D:\intellij project\cherry-in-the-haystack\`)에서 직접 수행한다.

---

## 3. 읽기는 자유, 수정은 허락 필수

- 파일 읽기(Read), 코드 탐색(Grep/Glob)은 자유롭게 수행한다.
- **파일 수정(Edit/Write), 명령 실행(Bash), 커밋(git) 등 변경을 수반하는 모든 작업은 반드시 사용자에게 먼저 허락을 받고 진행한다.**
- 허락 없이 임의로 코드를 수정하지 않는다.
